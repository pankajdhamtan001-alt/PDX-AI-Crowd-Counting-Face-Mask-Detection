# 🚀 PDX AI - Crowd Counting & Face Mask Detection

An AI-powered real-time Crowd Counting and Face Mask Detection system built using **YOLOv8**, **FastAPI**, **OpenCV**, and **PyTorch**. The application supports image, video, and live webcam processing with real-time analytics.

---

## 📌 Features

- 👥 Real-time Crowd Counting
- 😷 Face Mask Detection
- 📷 Image Upload Detection
- 🎥 Video Upload Detection
- 📹 Live Webcam Detection
- ⚡ FastAPI REST APIs
- 🔄 WebSocket-based Live Streaming
- 📊 Real-time Analytics
- 🎯 Adjustable Confidence Threshold
- 🖥️ User-Friendly Web Interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/pankajdhamtan001-alt/PDX-AI-Crowd-Counting-Face-Mask-Detection.git
```

Move into the project folder

```bash
cd PDX-AI-Crowd-Counting-Face-Mask-Detection
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```

The application will start on:

```
http://127.0.0.1:8000
```
